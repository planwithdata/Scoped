Sustainability Score Calculation (0-100):

Comprehensive scoring across 7 categories:

  • Green Spaces (20%): 15+ parks/forests within 1km
  • Air Quality (20%): AQI scale from excellent (25) to hazardous (200+)
  • Weather (10%): Temperature ~25¬∞C, humidity ~50%

  • Services (10%): 20+ banks/markets/police within 1.5km

Perfect Score (100): Optimal levels in all 7 categories

Data Sources:

  • Green spaces: OpenStreetMap via Overpass API
  • Air quality: Estimated based on location
  • Elevation: Open-Elevation API
